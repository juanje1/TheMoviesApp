# TheMoviesApp
Application for showing a movies list and the detail of each movie

## Description
The application has the following characteristics:

- Kotlin Language
- Jetpack Compose
- Clean Architecture
- MVVM Pattern
- Dagger Hilt
- Retrofit
- OkHttpClient
- Coroutines
- Flow, StateFlow
- Room
- Navigation Compose
- Unit Testing
- Request to an API
- Pagination
